What is Quick Search Popup?

Quick Search Popup is a lightweight Chrome extension that lets you search the web instantly without leaving the page you're on. Press Shift twice and a sleek search overlay appears — no mouse clicks, no tab switching, just type and go.

Inspired by the classic Opera browser's quick-dial search, rebuilt for the modern web with AI-powered search engines, full customization, and a clean keyboard-first experience.

✨ Features
  • ⚡ Instant overlay — Double-tap Shift (or custom shortcut) to open from any page
  • 🤖 AI Search built-in — Perplexity AI, ChatGPT, Claude ready to use
  • 🖱️ Mouse wheel engine switch — Scroll on the panel to cycle search engines
  • 📝 Auto-fill selected text — Highlight text on a page, then trigger to search it instantly
  • 🕐 Search history — Recent searches appear as you type
  • 🎨 Full theme control — 9 preset colors + custom hex per engine
  • 🌐 13 languages supported — Korean, English, Japanese, Chinese, and more
  • 🔒 Zero data collection — All settings stored locally. No servers, no tracking.
🚀 How to use
  1. Press Shift twice quickly on any webpage
  2. Type your search query (or it auto-fills if you had text selected)
  3. Use ↑ ↓ or mouse wheel to switch engines
  4. Press Enter to search · Esc to close
