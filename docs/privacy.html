⚡ Privacy Policy

Quick Search Popup — v1.0.0

Last updated: March 2026

Overview

Quick Search Popup is a browser extension that provides an instant search overlay on any webpage. We are committed to protecting your privacy. This extension does not collect, store, or transmit any personal data.

Data We Do NOT Collect

  • Personal identification information
  • Browsing history or visited URLs
  • Search queries entered in the popup
  • Keystrokes or user activity
  • Location data
  • Any data from the web pages you visit

Local Storage Only

The extension uses Chrome's storage API to save your preferences locally on your device only. This includes:

  • Selected search engines and their order
  • UI language preference
  • Color theme settings
  • Keyboard shortcut configuration
  • Recent search history (stored locally, never transmitted)
  • New tab preference

This data never leaves your device and is never sent to any server, including ours.

No Third-Party Data Sharing

We do not sell, trade, or transfer any data to third parties. When you perform a search, your query is sent directly to the search engine you selected (e.g. Google, Naver, ChatGPT) — exactly as it would be if you searched from your browser's address bar. Quick Search Popup does not intercept or log these queries.

No Remote Code

This extension does not load or execute any remote code. All functionality is contained within the locally installed extension package. No external scripts, CDNs, or analytics libraries are used.

Permissions

The extension requests the following permissions:

  • storage — Used solely to save your preferences locally. No data is transmitted externally.
  • Host permissions (<all_urls>) — Required to display the search popup overlay on any webpage you visit. The extension does not read, modify, or transmit any page content.

Children's Privacy

This extension does not knowingly collect any information from children under the age of 13.

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top of this page.

Contact
